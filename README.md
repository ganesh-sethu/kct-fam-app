## Installation steps

- Install nodejs and npm
- clone the repo
- Navigate to server directory  `cd server`
- `npm install`
- `nodemon`
- Navigate to client directory  `cd client`
- `npm install`
- `npm start`



Go to <http://localhost:3000>
